Add fields for title and meta description on post, pages, categories, tags and WooCommerce products. Light weight!

== Description ==

Joca SEO makes it really simple to make your Wordpress site more search engine friendly. It's simple to work with and has very few settings.

Joca SEO is a fast and light weight (<30 kb) plugin with no jQuery or external domain library, keeping your web site optimized.

Not tested on old WordPress versions, but we see no reason why it would not work on them so feel free to test it on any version you want.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/joca-seo` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add title and meta description info on posts, pages and WooCommerce products.


== Frequently Asked Questions ==

= Can I import data from my current SEO plugin? =

Yes, if you have the pro version of Joca SEO and have used Wordpress SEO (Yoast) then you can import all the meta data into Joca SEO.
